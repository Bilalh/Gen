Requires
	
	Python3.4
	ghc 7.8 to compile
	Compatible Conjure, Savilerow & Minion

To build (places `testSample` in ./bin)

	export CONJURE_REPO=<where conjure repo is>
	./make.sh


Before running

	export PARAM_GEN_SCRIPTS=<full path of instancegen/scripts>


		
Usage from 
	
	testSample --help
	
wrapper 
	
	testSampleWrap.sh
